# A Simple Chat Application using NodeJS, JQUERY and Laravel Framework
#installation Instructions
 - Laravel    
    -- Go to SimpleChat Folder 
        $ composer install
        $ npm install
        $ npm run dev
        $ php artisan migrate
        $ php artisan serve
 - Start MySQL Server
 - Nodejs 
   -- Go to SimpleChat->Socket Folder 
        $ npm install        
        $ nodemon index.js or node index.js
  >
